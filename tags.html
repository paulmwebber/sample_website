Important HTML Tags

This page is designed as a quick refernce for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header include a table header
td table data include a table data cell
ol ordered list include an orderd list
ul unordered list include an unordered list
li list item include a list item

Inline Elements

Tag Name Purpse Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make text strong <strong>really important</strong> really important
a anchor make a hyperlink <a href="www.example.com">link</a> link
code code show text is programming code <code>html</code> html
img image include an image <img src="http://bit.ly/1MZAFuQ" alt="Michael Hartl">
